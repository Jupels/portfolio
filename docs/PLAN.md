# План разработки сайта-портфолио

## Обзор проекта
Сайт-портфолио продуктового дизайнера Александра Лыгина.
- Главная страница: информация + превью кейсов
- Страницы кейсов: детальное описание проектов (позже)

## Технологии
- **HTML5** — семантическая разметка
- **CSS** — vanilla CSS с переменными (без Tailwind)
- **JavaScript** — vanilla JS для интерактивности
- **Данные** — JSON файлы для мультиязычности

## Брейкпоинты
| Название | Ширина | Примечание |
|----------|--------|------------|
| Mobile   | 360px+ | Основной мобильный |
| Tablet   | 640px+ | Планшет/большой телефон |
| Desktop  | 1080px+ | Десктоп |
| Wide     | 1440px+ | Широкий экран (контент max-width: 1040px) |

## Дизайн-токены

### Шрифты
```css
--font-family: 'Inter', sans-serif;
--font-weight-regular: 400;
--font-weight-medium: 500;
--font-weight-semibold: 600;
```

### Цвета (Light theme)
```css
--color-primary: #000000;        /* Основной текст */
--color-secondary: #7c7c7c;      /* Вторичный текст */
--color-accent: #ff5900;         /* Акцент (оранжевый) */
--color-bg: #ffffff;             /* Фон страницы */
--color-bg-card: #f2f2f2;        /* Фон карточек */
--color-telegram: #34ace1;       /* Кнопка Telegram */
--color-button-dark: #0d0d0d;    /* Тёмная кнопка */
--color-muted: #b4b4b4;          /* Приглушённый текст */
```

### Цвета (Dark theme — заготовка)
```css
--color-primary: #ffffff;
--color-secondary: #a0a0a0;
--color-accent: #ff5900;
--color-bg: #121212;
--color-bg-card: #1e1e1e;
--color-telegram: #34ace1;
--color-button-dark: #ffffff;
--color-muted: #666666;
```

## Структура страницы

### Header (64px)
- Левая часть: CV, LinkedIn, Dribbble
- Центр: Логотип (36x36px)
- Правая часть: Блог, ТГК, Почта, RU/EN

### Hero Section
- Заголовок с описанием (max-width: 600px)
- Кнопка "Написать в тг"
- Подпись про опыт
- Логотипы компаний (горизонтальная лента)

### Content (кейсы)
- Карточки с закруглёнными углами (20px)
- Заголовок + описание + кнопка-стрелка
- Изображение кейса
- Некоторые с плашкой "Coming soon"

### Footer (100px)
- Копирайт: "Alex Lygin © 2026"
- Версия: "Версия 1.0"
- На мобилке: меню навигации

## Структура файлов
```
alexlgncom/
├── index.html              # Главная страница
├── css/
│   └── styles.css          # Vanilla CSS + переменные для тем
├── js/
│   └── main.js             # Смена языка, темы, интерактив
├── data/
│   ├── ru.json             # Русские тексты
│   └── en.json             # Английские тексты
├── img/
│   └── home/               # Картинки кейсов
├── icons/                  # Иконки (включая Coming_soon.svg)
├── logos/                  # Логотипы компаний
├── docs/
│   └── PLAN.md             # Этот план
└── log.md                  # Лог изменений
```

## Мультиязычность
- Данные хранятся в `data/ru.json` и `data/en.json`
- При загрузке: проверяем localStorage → по умолчанию "ru"
- При клике на RU/EN: меняем язык без перезагрузки
- Используем `data-i18n="key"` атрибуты на элементах

## Темы (подготовка)
- CSS переменные в `:root` (light) и `[data-theme="dark"]`
- Переключатель добавим позже
- По умолчанию: light theme
